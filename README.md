# Rust Dedicated Server (Windows)

Простой старт Rust сервера под Windows.

## Возможности
- Один `start.bat`
- Автоматически скачивает SteamCMD
- Автоматически обновляет Rust
- Работает на любом диске
- Подходит для новичков

## Как запустить
1. Скачать архив или репозиторий
2. Распаковать в любую папку
3. Открыть `start.bat`
4. Изменить верхний блок настроек
5. Запустить `start.bat`

## Если сервер не виден
Попробуйте поменять местами:
- `QUERY_PORT`
- `RCON_PORT`

Это зависит от сети / хостинга.

## Требования
- Windows 10 / 11
- Открытые порты (UDP / TCP)
